# Installing Flarial

1. Download the latest Flarial installer from our official website
2. Run the installer as administrator
3. Follow the installation wizard prompts
4. Launch Minecraft and look for the Flarial button in the main menu

Note: Make sure you have Minecraft Bedrock Edition installed before installing Flarial.
